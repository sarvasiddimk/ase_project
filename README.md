# Red Panther: Adaptive Service Ecosystem (ASE)

The **Red Panther ASE** is a cloud-based SaaS platform designed to transform automotive service operations by focusing on **Trust**, **Efficiency**, and **Capability**.

## Architecture

This project is a monorepo containing:

*   **apps/web**: Next.js Frontend (Customer Portal, Technician App, Advisor Dashboard).
*   **apps/api**: NestJS Backend (API Gateway & Microservices).
*   **packages/ui**: Shared UI components (Tailwind CSS).
*   **packages/config**: Shared configuration (ESLint, TSConfig).

## Getting Started

### Prerequisites

*   Node.js (v18+)
*   pnpm (recommended) or npm

### Installation

```bash
npm install
```

### Running Locally

```bash
npm run dev
```
